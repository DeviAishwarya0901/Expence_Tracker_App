                 EXPENSE TRACKER APP
    A simple app to help you track and manage your daily expenses using Streamlit.
    
  * FEATURES
* Add expenses (category, amount, description, date)
* View charts to understand spending
* See monthly expense trends
* Check expenses by category
* Filter expenses by month or category
* Saves data using a JSON file

*HOW TO RUN THE APP
  
  *Install required packages:
  
  pip install -r requirements.txt
  
  *Start the app:
  
  streamlit run app.py
  
  *Open browser and visit:
  
   http://localhost:8501

   *HOW TO USE
   
   *Add Expense: Use the sidebar to enter new expenses

*View Summary: See total and monthly expenses

*Analyze Data: View charts for better understanding

*Filter Data: Choose category or month to filter

*Delete Expense: Turn on delete mode to remove entries


    * EXPENSE CATEGORIES 
* Food
* Transportation
* Shopping
* Bills
* Entertainment
* Health
